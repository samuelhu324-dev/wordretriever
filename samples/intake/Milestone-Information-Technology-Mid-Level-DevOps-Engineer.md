About the Role Milestone IT is partnering with a large blue-chip client to find a Mid-Level DevOps Engineer who is ready to take the next step. You won’t just be "fixing servers"—you’ll be building automated, self-healing infrastructure.
You’ll join an engineering culture that treats infrastructure as code (IaC) and values developers who think like operators (and vice versa).
The Day-to-Day
CI/CD Mastery: Build and maintain seamless deployment pipelines to increase release velocity.
Infrastructure as Code: Use tools like Terraform or CloudFormation to manage environments.
Cloud Orchestration: Manage containerized workloads using Kubernetes or Docker.
Monitoring & Alerting: Proactively ensure system health using Prometheus, Grafana, or ELK stacks.
Security First: Implement "DevSecOps" practices to keep the platform bulletproof.
What You’ll Bring
Experience: 2–4 years in a DevOps or SRE-focused role.
Cloud Savvy: Strong hands-on experience with AWS, Azure, or GCP.
Automation Mindset: Proficiency in scripting (Python, Bash, or Go).
Tooling: Experience with CI tools (Jenkins, GitLab CI, or GitHub Actions) and IaC (Terraform/Ansible).
Communication: The ability to collaborate across dev teams to break down silos.
The Rewards
Competitive daily rate
Structured career path toward Senior/Lead DevOps roles.
Flexible hybrid model 
Budget for certifications (AWS/CKA) and training.
